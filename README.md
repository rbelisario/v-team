# 🏐 VTeam

Aplicativo mobile para organização e formação de equipes de vôlei.

---

## 📱 Sobre o projeto

O VTeam tem como objetivo facilitar a criação de equipes equilibradas e o agendamento de partidas de forma simples e eficiente.

---

## 🚀 Tecnologias

* React Native
* Expo
* TypeScript
* Node.js (futuro backend)

---

## 📦 Estrutura do projeto

```
apps/
  mobile/

packages/
```

---

## 🧠 Arquitetura

O projeto utiliza **monorepo**, permitindo:

* compartilhamento de código
* escalabilidade
* separação entre frontend e backend

---

## 📋 Funcionalidades

* [ ] Cadastro de jogadores
* [ ] Formação de equipes
